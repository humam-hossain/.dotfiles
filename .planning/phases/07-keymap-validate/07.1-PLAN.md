# Phase 7.1: Fix Git Commits Picker

**Goal:** Add `<leader>gc` keymap for git commits/changes picker

**Gap Closure:** Closes UAT gap from Phase 7 keymap validation

---

## Tasks

### 7.1.1: Add `<leader>gc` to Registry

**File:** `.config/nvim/lua/core/keymaps/registry.lua`

Add entry to `M.lazy` section under Git domain (after line 460):

```lua
  {
    id = "git.commits",
    lhs = "<leader>gc",
    mode = "n",
    desc = "Browse git commits",
    domain = "g",
    scope = "lazy",
    plugin = "nvim-neo-tree/neo-tree.nvim",
    action = ":Neotree git_commit<CR>",
  },
```

### 7.1.2: Verify Keymap

Run `nvim` and confirm `<leader>gc` opens neo-tree git commits view.
