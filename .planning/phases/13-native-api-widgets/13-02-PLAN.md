## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Quickshell ↔ Hyprland IPC | `Hyprland.dispatch` calls cross from QML into the compositor's command parser. |
| Quickshell ↔ PipeWire | AudioService writes to `.audio.volume`/`.audio.muted` on the system audio sink. |
| Quickshell ↔ pavucontrol launch | `Process { command: ["pavucontrol"] }` spawns a system binary. |
