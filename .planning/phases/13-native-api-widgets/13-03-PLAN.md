|-----------|----------|-----------|-------------|-----------------|
| T-13-MPRIS-01 | Injection | MusicWidget MPRIS string handling | mitigate | `trackArtist`/`trackTitle`/`trackAlbum` are bound ONLY into `Text.text` and `ToolTip.text`. No `Process` block exists in MusicWidget; no `Hyprland.dispatch` call exists. Verified by negative grep in acceptance criteria. |
